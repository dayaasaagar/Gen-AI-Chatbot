# 🧠 GenAI Chatbot API

This project is a **domain-specific chatbot backend** built using:
- 🔥 FastAPI for API serving
- 🤗 Hugging Face's SentenceTransformers for embeddings
- 🧪 FAISS for fast vector similarity search
- 🧠 OpenAI GPT (like `gpt-3.5-turbo`) for response generation
- 📚 RAG (Retrieval-Augmented Generation) pipeline to enhance accuracy

---

## 🚀 Features

- ⚡ Fast & lightweight API using FastAPI
- 🔍 Retrieves relevant documents from a knowledge base using semantic search (FAISS + embeddings)
- 🤖 Combines context + query to generate accurate answers via OpenAI GPT
- 🔄 Easily extendable for your own domain-specific use cases
